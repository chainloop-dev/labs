# c8l cli get

[g] Get artifact from Chainloop

| Attributes       | &nbsp;
|------------------|-------------
| Alias:           | g

## Usage

```bash
c8l cli get [SHA]
```

## Examples

```bash
cl g 01430ba1c938528a13cea49cd62862c4820e3edc6fc92a9ac5a7f007ea5478ba
```

## Arguments

#### *SHA*

SHA256 of the artifact to get.
