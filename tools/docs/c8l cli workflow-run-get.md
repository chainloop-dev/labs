# c8l cli workflow-run-get

[wrg] Get workflow run from Chainloop.

| Attributes       | &nbsp;
|------------------|-------------
| Alias:           | wrg

## Usage

```bash
c8l cli workflow-run-get [UUID]
```

## Examples

```bash
cl wrg be52251b-0d8a-4719-bc20-16a586555ea4
```

## Arguments

#### *UUID*

UUID of the workflow run to get.
